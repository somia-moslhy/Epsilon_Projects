## 🌟 Highlight: The Survival Game Strategy
This is a multi-player strategy game built with pure Python logic.
* **Inventory System:** Optimized using Python **Dictionaries** for O(1) access speed.
* **Auction Mechanism:** Players bid coins to win items based on daily needs.
* **Dynamic Difficulty:** The game gets harder as days progress (Damage Scaling).
* **Code Structure:** Clean code principles with separated logic (`functions.py`) and execution (`main.py`).
